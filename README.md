## Описание программы
Book Tracker - это графическое приложение на Python для управления списком прочитанных книг. Программа позволяет добавлять, удалять и фильтровать книги, а также сохраняет данные в формате JSON.

### Основные возможности:
- Добавление книг с указанием названия, автора, жанра и количества страниц
- Просмотр всех добавленных книг в удобной таблице
- Фильтрация книг по жанру и количеству страниц
- Удаление книг из списка
- Автоматическое сохранение данных в JSON файл
- Загрузка данных при запуске программы
- Валидация вводимых данных

## Требования к системе
- Python 3.6 или выше
- Операционная система: Windows, macOS или Linux
